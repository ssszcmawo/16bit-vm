# 16bit-vm

This project implements a simple 16-bit virtual machine with registers, memory, and an instruction set.
The emulator simulates the operation of a processor with basic system calls and arithmetic logic.

This project was created following this guide:
https://www.andreinc.net/2021/12/01/writing-a-simple-vm-in-less-than-125-lines-of-c#the-registers
